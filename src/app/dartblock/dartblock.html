Technical details: DartBlock has been written using the Flutter framework, though it can be integrated within any web app as an iframe by building the Flutter app for the Web platform. However, the Web version comes with a limitation compared to the other supported platforms: due to the lack of support for isolates, programs are executed on the same thread on which the Flutter app itself is running. For more details, see the GitHub page.
